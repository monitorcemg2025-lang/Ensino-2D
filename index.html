Crie Seu Primeiro Jogo 2D no Godot 4

Tutorial completo em português – Dodge the Creeps

Bem-vindo(a)!

Neste tutorial você vai criar um jogo 2D completo onde o jogador se move e evita inimigos que aparecem na tela.

Tempo estimado: 1–2 horas | Nível: Iniciante

Baixar Godot 4 (Grátis) Baixar Assets do Jogo

Passo a Passo Completo

  1. Criar o projeto → New Project → 2D → Nome: DodgeTheCreeps
  2. Configurar janela → Project Settings → Display → Window → Size: 480x720
  3. Criar Player → CharacterBody2D + Sprite2D + CollisionShape2D
  4. Script do Player (GDScript completo incluso abaixo)
  5. Criar Inimigo (Mob) → RigidBody2D com animação
  6. Spawn aleatório com Path2D + Timer
  7. HUD com pontuação e Game Over
  8. Testar e exportar!

Script do Player (player.gd)

extends CharacterBody2D

@export var speed: int = 400
var screen_size: Vector2

func _ready():
    screen_size = get_viewport_rect().size

func _physics_process(delta):
    var direction = Input.get_vector("move_left", "move_right", "move_up", "move_down")
    velocity = direction * speed
    move_and_slide()
    
    # Limitar à tela
    position = position.clamp(Vector2.ZERO, screen_size)
